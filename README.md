Fun viking game where you have to advance through levels kill bosses, get powerups, which currently is playable with rppg technology (webcam/phone camera), brain devices like Muse to use biosignals in app.

It will be playable with sentiment analysis, and BCI functionality moving forward as well.

V.0.6 is in progress and about to be redeployed to the closed beta of the Elata app store (https://app.elata.bio/) with functional rppg and brain device support. Let me know if you want an early access code and I will hook you up.
